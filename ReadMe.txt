

In order to run the programme, you need to create the following tables in database--

1.create table emp(name varchar(20),age number(3),job varchar(20),salary number(8),email varchar(20),pno number(15),username varchar(20),password varchar(10),id number(10));
2.create table payment(incoming varchar(20),out varchar(20),dates date,sessiontime varchar(20),payment varchar(20));
3.create table admin(username varchar(20),password varchar(20));

Ensure that Password of the database is set to '1234'.
